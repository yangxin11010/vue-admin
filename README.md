# vue-admin

## Project setup

```
git clone https://gitee.com/yangxin11010/vue-admin.git
or
git clone https://github.com/yangxin11010/vue-admin.git


cd vue-admin

npm install
or
yarn / yarn add
```

### run

```
npm run dev
or
yarn dev
```

### build for development

```
npm run build-dev
```

### build for production

```
npm run build
```
